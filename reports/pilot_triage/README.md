# Pilot Triage Summaries

This directory is for local MVP Pilot issue triage summaries generated by:

```bash
uv run python scripts/phase237d_pilot_triage_summary.py --input-dir reports/pilot_issues
```

Real summary JSON / Markdown files are local trial artifacts and are ignored by Git by default.

Do not treat these summaries as repair results, rollout approval, external issue creation, or automatic tender-review conclusions.
