# MVP Pilot Review Reports

This directory is reserved for local MVP Pilot review dry-run reports.

Real report JSON and Markdown files are ignored by default and must not be committed to Git.

These reports are for human review only. They are not production rollout approval, not repair authorization, and not automatic bid-review or business-decision output.

Do not store secrets, raw sensitive customer content, unredacted business judgment, or non-sanitized client data in this directory.
